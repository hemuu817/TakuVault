TRPGセッションツール向け、クラウド素材保管庫「TakuVault」

##サービス概要

TakuVault は、TRPGセッションツールで使用した背景・カットイン・BGM等の素材をクラウド上に保存し、
「どのセッション / どのシーン / どの役割で使った素材か（Where used）」を逆引きできる素材保管庫です。
※MVPでは Cocofoliaのみ を対象にし、部屋連携は room_urlの参照保存（同期やAPI連携は行わない）とします。

##このサービスへの思い・作りたい理由

趣味のTRPGのKPとして部屋を作る中で、使用した素材を手元で雑に管理してしまい、
「いつ・どのシーンで使った素材か分からない」「良かったBGMや背景が見つからない」という問題が頻発しました。
既存ツールにも素材保存機能はある一方、シーン単位での分割管理 / リネーム / タグ付け / 逆引き検索には不便を感じています。
そこで、複数セッション・複数ツールに跨って素材を整理できる「クラウドの素材保管庫」を作りたいと考えました。

##ユーザー層について
- TRPGのKP/GMで、セッション準備の素材管理を効率化したいユーザーをターゲットにしています。
- ライトユーザー：素材をクラウドに分かりやすくまとめておきたい
- ヘビーユーザー：複数セッション/複数ツールの素材を一元管理したい

##サービスの利用イメージ
- セッションのURLリンクを貼り付けることで、アプリケーションからセッションにリダイレクトできます。
- 投下した素材に任意で リネーム / タグ付け ができます。
- 元のファイル名または付与した名前で検索でき、タグの属性による絞り込みも行えます。
- 保存した素材は、使用したツール・セッションを跨いで ドラッグ&ドロップで移動できます。
※ドラッグ&ドロップはMVPに含めるか状況により調整します（導線はMVPでも必須）

##ユーザーの獲得について
- SNS（X など）でTRPG関連のハッシュタグを用いて発信し、特にセッション準備を行うKP/GM層に向けて、以下を主軸に訴求します。
- セッション素材（背景/カットイン/BGM等）を 一元管理できること
- 「どのセッション / どのシーン / どの役割で使ったか」を追跡できること（Where used）
- 次回以降のセッション準備における 検索・再利用の効率化

##サービスの差別化ポイント・推しポイント

TakuVaultの最大の特徴は、素材を単に保存するだけでなく、
Where used（素材がどのセッション/シーン/役割で使われたか）を記録し、素材側から逆引きできる点です。
このアプリケーションでは、「あの時の背景/BGMどれだっけ？」の疑問を即座に検索・再利用して解決できます。
また、将来的には複数のセッションツール（Cocofolia以外）にも対応し、ツールを跨いだ素材管理を可能にすることで継続利用の価値を高めていく想定です。

##機能候補
- MVPリリース時点では、対応するセッションツールは Cocofoliaのみ とします。
- ルーム連携は同期ではなく、Cocofoliaの部屋URL（room_url）を保存して参照できる形に留めます。
- MVPでは、素材（Asset）と使用情報（Usage）を中心に、Where used の価値が成立する最小機能を実装します。

## 今後の展望
MVPリリース後の開発方針として、以下の機能追加をしていく方針です。
- 素材種別の拡張：背景/カットイン/BGM以外（SE、立ち絵、マップ、キャラ駒等）
- 登場人物・シーン分類などの属性追加
- お気に入り機能などの保存検索機能の追加
- 動画→APNG+MP3変換（非同期処理＋進捗表示）の追加

##使用する技術スタック
- 使用するフレームワーク: Ruby 3.x, Rails 7.x（フルスタック）
- データベース: PostgreSQL
- デプロイ先: Render（Web：Dockerデプロイ）
- ストレージ / ライブラリ: ActiveStorage, 本番ストレージ：Cloudflare R2（S3互換）を利用予定
